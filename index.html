MINHAS PRINCIPAIS HABILIDADES.

HTML

Com o desenvolvimento dos navegadores e a popularização da internet, o HTML passou a ser utilizado na criação de sites. Atualmente, a tecnologia evoluiu e é utilizada em conjunto com outras ferramentas no desenvolvimento de sites e aplicativos web dinâmicos.

CSS

O CSS permite modificar o visual de vários elementos ou seções, e além de adicionar animações e outros recursos visuais, também é possível determinar que os elementos sejam renderizados de maneira diferente, de forma que se adequem a diversos dispositivos.

JavaScript

Se trata de uma das principais linguagens de programação, oferecendo uma grande variedade de frameworks e aplicações para os mais diversos dispositivos, e se destaca por deixar as páginas Web dinâmicas, o que é sua principal função.

TypeScript

O TypeScript é uma linguagem criada em cima do JavaScript e permite que adicionemos tipagem de dados em nossas variáveis, além de nos oferecer outros recursos avançados, aumentando o horizonte de possibilidades do uso desta linguagem.

Node.js

É um ambiente de execução que permite executar aplicações desenvolvidas com a linguagem JS de forma autônoma. Com ele, é possível criar praticamente qualquer tipo de aplicação web, como sites estáticos ou dinâmicos, APIs e sistemas baseados em microserviços.

React

É uma das bibliotecas de JS mais populares para desenvolvimento de aplicativos web ou para dispositivos móveis, contendo uma coleção de trechos de código de JavaScript reutilizáveis chamados componentes, usados para a construção da interface do usuário.
