MY MAIN SKILLS.

HTML

With the development of browsers and the popularization of the internet, HTML began to be used in website creation. Today, the technology has evolved and is used together with other tools in the development of dynamic websites and web applications.

CSS

CSS allows for modifying the appearance of various elements or sections, and in addition to adding animations and other visual features, it also enables elements to be rendered differently, adapting to various devices.

JavaScript

JavaScript is one of the main programming languages, offering a wide variety of frameworks and applications for various devices. It stands out for making web pages dynamic, which is its primary function.

TypeScript

TypeScript is a language built on top of JavaScript that allows us to add data typing to our variables and offers other advanced features, expanding the possibilities for using this language.

Node.js

Node.js is a runtime environment that allows for executing applications developed in JavaScript independently. With it, it's possible to create virtually any type of web application, such as static or dynamic websites, APIs, and systems based on microservices.

React

React is one of the most popular JavaScript libraries for developing web or mobile applications. It contains a collection of reusable JavaScript code snippets called components, used to build the user interface.
